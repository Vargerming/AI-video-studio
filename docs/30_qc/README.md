# QC Baseline

## Core rule
- Fragment QC: review in pieces <= 30 seconds
- Batch management: group fragments, track status
- Final combined review after merge
- Operator notes must be stored per fragment and for final review

## Pass/Fail (draft)
PASS if:
- audio intelligible
- subtitles aligned
- no obvious visual artifacts
- segment matches script intent

FAIL if:
- missing audio/subtitles
- severe artifacts
- wrong language/voice
- continuity breaks that affect meaning
